## Installation

```bash
# 1) create and activate virtual environment
# -- EITHER with conda
conda create -n apis_env python=3.11
conda activate apis
# -- OR with venv
python3.11 -m venv apis_env
source apis_env/bin/activate

# 2) install dependencies
(apis_env) pip install -r requirements.txt
```
